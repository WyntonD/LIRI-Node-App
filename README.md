# LIRI-Node-App

Bandsintown

<h1>How it works</h1>
 <li> Clone the repository </li>
 <l1> Enter the repo and "npm install node" </li>
 
 <h1> Commands </h1>
 <h2> To search for concerts <h2>
 "node liri.js concert-this (music artist name)"
